🧠 課題内容（提出必須）

課題①：じゃんけんプログラムを作ろう

仕様：
	1.	コンソールに「じゃんけんをしましょう」と表示
	2.	プレイヤーが 0:グー 1:チョキ 2:パー を入力
	3.	コンピュータはランダムで0〜2を出す
	4.	勝敗を判定して結果を表示する（例：「あなたの勝ち！」「あいこです」など）
	5.	勝敗判定をメソッドとして分離する（例：judge(int player, int cpu)）

実行例：

じゃんけんをしましょう
0:グー 1:チョキ 2:パー ＞ 0
あなた：グー　コンピュータ：チョキ
あなたの勝ち！


⸻

課題②：改良版じゃんけん（チャレンジ課題）

下記のいずれかを自分で選んで実装してください。
	•	勝敗をカウントして「3回勝負」にする
	•	入力値が0〜2以外のときに「再入力」を促す
	•	文字で入力（例：「グー」や「パー」）でも動くようにする
